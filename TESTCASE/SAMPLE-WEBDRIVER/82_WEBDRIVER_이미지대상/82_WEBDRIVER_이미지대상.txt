;-----------------------------------------------------------------------------------------------------------------------------------------
;# 설명 : WebDriver 테스트시 이미지와 Xpath 방식을 서로 혼용 사용 가능함.
;-----------------------------------------------------------------------------------------------------------------------------------------

"$신규세션={host=localhost:4444,browserName=chrome,platform=WINDOWS}" 세션생성 한다.
네이버 지도 메인 http://www.navercorp.com/ko/index.nhn 으로 접속한다. 

; 이미지 방식으로 확인
네이버로고  확인한다.

; 하단에 위치해서 보이지 않는 이미지로 화면을 스크롤 하기 위해 xpath 형태로 주변 대상을 확인함  (xpath 형태로 '확인'시 해당 위치로 화면이 자동 스크롤됨)
"{xpath://a[contains(.,'기업윤리상담센터')]}"  확인한다.

; 하단으로 스크롤된 상태에서 이미지를 찾아서 클릭
하단메뉴-개인정보처리방침 클릭한다.


; 이미지 방식으로 팝업 브라우저 선택
팝업창-개인정보처리방침 선택한다.
현재화면을 캡쳐한다.

; 팝업창 닫기 (세션 종료는 아니며, 팝업창만 종료됨)
현재창을 종료한다. 

; 계속 테스트를 진행하기 위해 이전 창을 다시 xpath 방식으로 지정함
네이버로고  선택
현재화면을 캡쳐한다.

브라우저를 세션종료한다.
