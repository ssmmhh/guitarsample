;# 설명 : 변수를 활용 테스트 (변수와 문자열의 조합, 변수 재사용) 
;-----------------------------------------------------------------------------------------------------------------------------------------

"$호스트정보=host=localhost:4444" 설정
"$신규세션={|$호스트정보|,browserName=chrome,platform=WINDOWS}" 세션생성 한다

;메인주서를 변수로 설정
$뉴스URL=http://news.naver.com 설정한다.

;ID=변수사용 테스트_001
;변수 활용, 변수와 문자열을 합치기 위해서는 "|" 기호를 사용한다. 
$뉴스메인URL=$뉴스URL|/main/home.nhn 설정 한다.
$뉴스메인URL으로 접속한다. 

;xpath 형태로 H1태그 "뉴스" 를 찾음
"$찾는태그={xpath://h1[contains(.,'뉴스')]}" 설정한다.
$찾는태그 확인한다.
현재 화면을 캡쳐한다. 


;ID=변수사용 테스트_002
;변수와 문자열을 합친 최종 URL로 접속한다. 
$뉴스URL|/sports/new/main/index.nhn 으로 접속한다. 

;$찾는태그 값을 변수로 저장하여 사용 
; "스포츠" 로고 링크를 확인 
"$찾는태그={xpath://a[@class='logo_sports']}" 설정한다.
$찾는태그 확인한다.
현재 화면을 캡쳐한다. 

브라우저를 세션종료한다.
