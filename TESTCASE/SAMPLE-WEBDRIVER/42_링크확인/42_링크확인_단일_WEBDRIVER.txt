;-----------------------------------------------------------------------------------------------------------------------------------------
;# 설명 : xpath 형태로 a태그 링크를 확인하는 예제
;-----------------------------------------------------------------------------------------------------------------------------------------

"$신규세션={host=localhost:4444,browserName=chrome,platform=WINDOWS}" 세션생성 한다.

http://map.naver.com 접속한다.

; 날씨 아이콘 클릭
"{xpath://a[contains(@href, 'townWetr.nhn')]}"를 클릭한다.

; 새로운 날씨창으로 작업 대상을 이동
"{xpath://a[text()='날씨 홈']}" 선택한다.

; 현재 열려진 날씨 팝업창을 종료
현재 창을 종료한다.

; 이전 창으로 포커스를 이동
"{xpath://a[contains(@href, 'townWetr.nhn')]}"를 선택한다.


;다른주소로 접속
http://map.naver.com 접속한다.

; 네이버 메인 화면의 링크정보 확인
"{xpath://a[@href='http://www.naver.com']}"를 확인한다.

브라우저를 세션종료한다.
