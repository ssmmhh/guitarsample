;-----------------------------------------------------------------------------------------------------------------------------------------

IE 생성한다.

; "나는가수다" 통합검색결과 페이지 
http://search.naver.com/search.naver?where=nexearch&query=%B3%AA%B4%C2%B0%A1%BC%F6%B4%D9&sm=top_hty&fbm=1 접속한다.

; 검색결과화면에서 검색어 일치여부 확인
"$검색어=[input:name=query:::value]"  속성읽기 한다.
$검색어=나는가수다 같지않으면 실패한다.

; 검색어를 변경 뒤 재검색 수행
"[input:title=검색어:::value]=너도가수냐?"  속성쓰기 한다.
"[button::검색]"  클릭한다.

2초 대기한다.

현재화면을 캡쳐한다.



http://help.naver.com/support/bbs/notice.nhn 접속한다.
; ^ 를 사용하여 and 옵션으로 필터를 지정 

; DIV 항목중 class 와 id 2개 조건을 만족하는 항목을 찾음 (상단 게시물 제목)
"[div:class=skin skin_del_writer_recommend^id=n2_no_recommend]" 확인한다.

현재화면을 캡쳐한다.

IE 종료 한다.
