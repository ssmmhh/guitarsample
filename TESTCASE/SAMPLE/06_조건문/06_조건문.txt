IE를 생성한다. 

;ID=조건문_ 예제1
네이버 지도 메인 http://www.naver.com/ 으로 접속한다. 

; 네이버로고 이미지 존재여부에 따라 특정 변수 값을 설정 하거나 명령을 수행함
$로고존재여부=True 설정한다.

네이버로고가 없으면 실패한다.
네이버로고가 있으면 2초 대기한다. 
네이버로고가 있으면 $로고존재여부=True 설정한다.

; 화면내의 Text 내용의 존재여부에 따라 명령 분기
$회원가입존재=False 설정한다.
화면에 "회원가입" 내용이있으면 변수 $회원가입존재=True 설정한다.
화면에 "로그아웃" 내용이없으면 변수 $로그인상태=False 설정한다.

; 로그인 상태일때 내소식이미지가 있는지 확인한다. 
로그아웃버튼이 있으면 내소식을 확인한다. 

; 로그인된 상태라면 로그아웃 상태로 변경한다.
로그아웃버튼이 있으면 로그아웃버튼을 클릭한다.

현재 화면을 캡쳐한다.


;ID=조건문_ 예제2
;# 변수값 설정 및 값비교에 따른 조건 분기
$로그인상태=비로그인 설정한다.

네이버 지도 http://www.naver.com 으로 접속한다. 
만약 로그아웃버튼이 있으면 "$로그인상태=로그인" 으로 설정한다.

변수값의 상태가 "$로그인상태=로그인" 같으면 실패한다.
변수값의 상태가 "$로그인상태=비로그인"  같지않으면 실패한다.

;ID=조건문_ 예제3

;# 변수 설정
$a=1 설정한다.
$b=2 설정한다.
$c=3 설정한다.

;# 단일 비교문
"$a=1"과 같으면  {
	$a=1 임으로 0.1초 대기한다.
	$a=1 임으로 0.2초 대기한다.
 } 

;# 다중 비교문
"$b=2"과 같으면  {
	$c 비교 전 0.1초 대기한다.
	$c=3 같으면 {
		$c=3 임으로 0.1초 대기한다.
	}
	$c=4 같으면 {
		$c=4 임으로 0.1초 대기한다.
	}
	$c 비교 후 0.1초 대기한다.
 } 

브라우저를 종료한다.
